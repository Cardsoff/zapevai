# Запевай 🎤

Тренажёр для заучивания текстов песен. Apple-стиль, мобильный-first, три режима тренировки, интервальные повторения, общая база песен с защитой от дублей.

## Что внутри

- **Пропуски слов** — 5 уровней сложности (10% → 100% скрытых слов), открытие тапом и самопроверка «Помнил / Забыл»
- **Первые буквы** — от каждого слова видна только первая буква (так учат тексты актёры)
- **Эстафета строк** — видишь строку, вспоминаешь следующую
- **Интервальные повторения** — система сама напоминает повторить песню через 1 → 3 → 7 → 14 → 30 дней
- **Серия дней, кольца прогресса, живой градиент, вибрация и звуки**
- **Гостевой режим** — без регистрации всё хранится в браузере; после входа — в облаке
- **Дедупликация** — одинаковые тексты (независимо от регистра, пунктуации и «ё») распознаются и не дублируются между пользователями

Стек: Next.js 14 · Tailwind CSS · Framer Motion · Supabase (PostgreSQL + Auth) · Vercel.

## Запуск и публикация — 3 шага (≈15 минут)

> Сайт работает и **без шагов 1–2** — в гостевом режиме (данные в браузере).
> Supabase нужен для регистрации и общей базы песен.

### Шаг 1. База данных — Supabase (бесплатно)

1. Зайди на [supabase.com](https://supabase.com) → **Start your project** → войди через GitHub или Google.
2. **New project**: имя `zapevai`, пароль базы — любой (сохрани), регион — Frankfurt (ближе к РФ).
3. Слева открой **SQL Editor** → вставь содержимое файла `supabase/schema.sql` → **Run**. Готово: таблицы и защита данных созданы.
4. Слева **Project Settings → API** — скопируй два значения:
   - `Project URL`
   - `anon public` ключ
5. (Необязательно) Вход через Google: **Authentication → Providers → Google** → включи и следуй подсказке. Вход по email работает сразу без настройки.

### Шаг 2. Код — GitHub

1. Зайди на [github.com](https://github.com) → **New repository** → имя `zapevai` → Create.
2. Нажми **uploading an existing file** → перетащи в окно всё содержимое этой папки (кроме `node_modules`, если есть) → **Commit changes**.

### Шаг 3. Хостинг — Vercel (бесплатно)

1. Зайди на [vercel.com](https://vercel.com) → войди через GitHub.
2. **Add New → Project** → выбери репозиторий `zapevai` → Import.
3. Раскрой **Environment Variables** и добавь две переменные из шага 1:
   - `NEXT_PUBLIC_SUPABASE_URL` = Project URL
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY` = anon public ключ
4. **Deploy**. Через ~2 минуты получишь ссылку вида `zapevai.vercel.app` — это и есть твой сайт.
5. В Supabase: **Authentication → URL Configuration** → в `Site URL` вставь адрес сайта (чтобы вход через Google возвращал на сайт).

### Локальный запуск (по желанию)

Нужен [Node.js](https://nodejs.org). В папке проекта:

```bash
npm install
npm run dev
```

Сайт откроется на http://localhost:3000. Ключи Supabase можно положить в файл `.env.local` (образец — `.env.example`).

## Лимиты бесплатных тарифов

Vercel и Supabase бесплатно тянут тысячи пользователей: у Supabase лимит 500 МБ базы и 50 000 активных пользователей в месяц — для первых 50–100 человек запас огромный.
